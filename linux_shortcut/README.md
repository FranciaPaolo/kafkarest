# kafkarest helper.sh
Utility shortcut to access kafka from linux commands.

Run

    ./helper.sh

## Possible commands  

To count messages in the topic:  
    topic.list


To count messages in the topic:  
    topic.count fast-messages

To create new topic:  
    topic.create fast-messages

Start Apache Kafka:  
    kafka.start

To check if Kafka is running:  
    kafka.isrunning
